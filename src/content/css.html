Animation & transition

transition: name duration timingFunction delay

button {
background-color: gray;
transition: background-color .5s linear, color 1s ease-in-out;
button::hover{
background-color: green;
color: gray; }
}
W tym przykładzie animujemy przejście kolorystyczne tła butttona z szarego na zielony oraz koloru fontu po najechaniu myszką. Przy czym klor tła zmienia się liniowo i trwa krucej natomiat font zmnienia kolor dynamiczniej.

button {
background-color: gray;
transition: background-color .5s linear, color 1s ease-in-out;
will-change: background-color;
button::hover{
background-color: green;
color: gray; }
}

Możemy użyć funkcji 'will change: background-color;', która zapewni przyśpieszenie sprzętowe dla dynamiczniej animacji. Nie możemy jednak jej nadużywać.
