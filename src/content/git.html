    Inicjowanie pracy z gitem

    • git config --global user.name--konfiguracja nazwy użytkonika
    • git config --global user.email--konfiguracja emaila użytkownika

    Commitowanie zmian

    • git status--sprawdzamy co jest do zacommitowania
    • git add .--dodaj pliki do zacommitowania
    • git commit -m'masage to commit'--commituj zmiany
    • git push--wypchnij zmiany githuba
    • git restore--kasuje wszystkie zmiany niezacommitowane

    Git Bash zmiana home directory

    Są dwa sposoby żeby zmienić folder w którym startuje Git Bash:

    • Pierwszy
    • C:\Program Files (x86)\Git\etc\profile--otwieramy ten plik w edytorze np VS Code
    • HOME='/c/homepage--gulp'--dodajemy taki wpis w tym pliku np. pod takim wpisem INFOPATH='/usr/local/info:/usr/share/info:/usr/info:/share/info'
    • cd ~--wpisujemy w bashu i zostaniemy przeniesieni do ustwionej w pliku ścieżki
    • Drugi
    • Mój Komputer>Zaawansowane ustawienia sytemu>Zmienne środowiskowe...--wchodzimy do edytora zmiennych środowiskowych
    • Nowa...>Nazwa = HOME, Wartość zmiennej = c:\ścieżka--dodajemy nową zmienną systemową o nazwie HOME (nie zmienna użytkownika). Teraz Bash odpali się w podanej lokalizacji
