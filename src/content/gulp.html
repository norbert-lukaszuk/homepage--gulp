Instalacja Gulpa

W terminalu wpisujemy npm install gulp-cli--global lub wersja krótsza npm install -g - gulp-cli - tą instalację wykonujemy raz na cały system i najlepiej z uprawnieniami administartora

Kiedy zaczynamy pracę w nowym projekcie npm install a przed ropoczęciem pracy nad projektem wchodzimy do kataklogu głównego projektu i piszemy gulp.

Struktura katlogów w projekcie

Pracę nad projektem prowadzimy w katalogu src nie tworzymy żadnych zmian w katalogu dist jest to katalog wynikowy naszej pracy i przy każdej zmianie w src wszystkie zmiany zostaną nadpisane. Katalog src zawiera podkatalogi assets, content, js, pwa, scss W katalogu asstes przechowujemy zdjęcia i inne pliki zewnętrzne. W contetnt są pliki html, natomiast w scss przechowujemy style.
