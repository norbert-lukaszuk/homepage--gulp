Node.js

Pobieramy i instalujemy noda ze stronki www.nodejs.org wybieramy wersję LTS.

Opdalamy terminal lub wiersz poleceń w Windowsie i sprawdzamy czy node się zainstalował poleceniem node -v po czym zostanie wyświetlona wersja zainstalowanego noda. Teraz z poziomu terminala możemy odpalać pliki .js poza konsolą w przeglądarce. Używamy do tego polecenia node np. wpisujemy "node script.js" .

NPM - Node Package Manager

Inicjalizacja menedżera odbywa się w katalogu projektu do którego należy wejść robimy to poleceniem
"npm init -y" jeżeli nie chcemy wypełniać właściwości lub "npm init". Dla każdego folderu trzeba przeprowadzić oddzielną inicjalizację.

W momencie inicjalizacji tworzy się folder node_modules są tam przechowywane pliki modułów tego folderu nie commitujemy ponieważ jest "ciężki i zawiera niepotrzebne elementy". Git ignoruje pliki i foldery zapisane w pliku .gitignore. Tworzymy taki plik w edytorze kodu, otwieramy go i w pierwszej linijce wpisujemy node_modules i pozostawiamy wolną linijkę kodu. Zapisujemy plik i od tej pory folder node_modules nie będzie uwzględniany przez gita. Opcja ta może być wykorzystywana do każdego elementu, którego nie chcemy commitować.

Razem z pakietami instalują się także pliki package-lock.json i package.json zawierają one informacje o zainstalowanych pakietach i ich autentyczności. Możemy skasować cały folder node_modules po czym możemy przywrócić poleceniem npm install .
